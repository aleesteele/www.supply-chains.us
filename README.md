2.  **What is supply-chains.us?**

    Supply-chains.us is a place to visualize information, stories, and facts about supply chains. While it focuses on the technology industry, it's aim is to spread into other areas, and other types of supply chains.

    This repository is meant to be a [open source](https://drive.google.com/drive/folders/1TjqSrj9WgEnd_mJl1WJ22BDK1fOmnfbu?usp=sharing). All of the data and text used on this site is available to comment and edit.
